### Тестовое задание Продуктового аналитика
**Кейс 1.** По ссылке находится Массив по продажам. Проанализируйте его, выявите проблемы, обозначьте выводы.

**Кейс 2.** По ссылке находится Массив данных по отдельной номенклатуре. Проанализируйте его, выявите проблемы, обозначьте выводы.

Тестовое задание оформите в одном файле, подпишите и укажите длительность выполнения (в минутах / часах).

Пришлите ссылкой на документ, проверьте доступ.
